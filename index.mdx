## Get started

<Steps>
  <Step title="Install the server">
    Clone the repository and install dependencies using Python or Docker.
    ```bash
    git clone https://github.com/Bellaposa/airplanes-live-mcp.git
    cd airplanes-live-mcp
    python -m venv .venv && source .venv/bin/activate
    pip install -r requirements.txt
    ```
  </Step>
  <Step title="Configure Claude Desktop">
    Add the server to your Claude Desktop configuration file so Claude can connect to it.
  </Step>
  <Step title="Start tracking">
    Ask Claude anything about live air traffic: "Show me aircraft near New York" or "Find flight BA387".
  </Step>
</Steps>

<Note>
  This project uses the [airplanes.live API](https://airplanes.live/api-guide/) which is provided for **educational and non-commercial use only**. Please review the API terms before use.
</Note>
