The Airplane Tracker MCP Server bridges AI assistants and live flight data. It exposes the [airplanes.live API](https://airplanes.live/api-guide/) as a set of tools that any MCP-compatible client — including Claude Desktop — can call directly from a conversation.

Ask Claude "Show me military aircraft" or "Find flight BA387" and it queries the live data feed, formats the result, and returns it inline. No browser switching, no manual API calls.

## What is MCP?

MCP (Model Context Protocol) is an open standard that lets AI clients call external tools. When you configure an MCP server, the AI client discovers the available tools and can invoke them on your behalf during a conversation. The Airplane Tracker MCP Server implements this protocol so Claude can access real-time aircraft data.

## Server implementations

The repository includes two server files:

| File | Library | Use case |
|------|---------|----------|
| `airplane_server.py` | `mcp[cli]` | Standard MCP server. Use this for Claude Desktop configuration and direct testing. |
| `airplane_server_fastmcp.py` | `fastmcp` | FastMCP server. Used by the Docker image and `fastmcp run` command. |

Both implementations expose the same 9 tools and connect to the same airplanes.live API endpoints.

## Available tools

| Tool | Description |
|------|-------------|
| `aircraft_by_hex` | Search by Mode S transponder hex ID |
| `aircraft_by_callsign` | Search by ICAO flight callsign |
| `aircraft_by_registration` | Search by tail number |
| `aircraft_by_type` | Search by ICAO type code (e.g., B738, A321) |
| `aircraft_by_squawk` | Search by squawk code, including 7700 emergency |
| `aircraft_near_position` | Find all aircraft within a radius of coordinates |
| `military_aircraft` | List all tracked military aircraft |
| `ladd_aircraft` | List law enforcement and security aircraft |
| `pia_aircraft` | List special interest and private aircraft |
