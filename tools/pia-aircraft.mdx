---
title: "pia_aircraft"
description: "Retrieve all live aircraft tagged as PIA (private, interesting, or special aircraft)."
---

Returns all aircraft currently airborne tagged as PIA (Private / Interesting / Anomalous) in the airplanes.live database. No parameters required.

PIA aircraft are those flagged as special interest — this category commonly includes private executive jets, government aircraft, and other notable flights that exhibit patterns of interest to aviation enthusiasts and researchers.

## Parameters

This tool takes no parameters.

## Example queries

Ask Claude:
- "Show special interest aircraft currently flying"
- "List all PIA aircraft"
- "What private or interesting aircraft are airborne?"

## Example response

```

🛩️ Type: G650
📍 Position: 25.2048, 55.2708
📏 Altitude: 41000 ft
⚡ Ground Speed: 520 knots
🧭 Track: 270°
🔖 Mode S Hex: 896abc
👁️ Last Seen: 2 seconds ago

...
```

## Notes

<Note>
  PIA aircraft may have limited identifying information available, as many have requested reduced data visibility in public tracking systems.
